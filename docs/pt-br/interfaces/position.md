# Posição (Position)

A interface de Posição é utilizada para definir coordenadas em um sistema de grade (grid), como a localização de um pedaço de terra no mapa ou a posição de um objeto.

| Propriedade | Tipo | Descrição |
| ----------- | ---- | --------- |
| x           | int  | Coordenada no eixo horizontal |
| y           | int  | Coordenada no eixo vertical |

## Exemplo de definição

```json
{
    "x": 1,
    "y": 1
}
```

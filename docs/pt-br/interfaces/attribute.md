# Atributo

A interface de atributo é uma definição - uma habilidade ativa ou passiva - de um atributo que pode ser adicionado a uma unidade ou construção.

| Propriedade  | Tipo | Descrição                     |
| ------------ | ---- | ----------------------------- |
| attribute_id | int  | Identificador do atributo     |
| value        | int  | Valor semente para o atributo |

## Exemplo de definição

```json
{
    "attribute_id": 1,
    "value": 50
}
```

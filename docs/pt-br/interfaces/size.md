# Tamanho

A interface de tamnaho é a definição da largura e o comprimento de um objeto, sendo uma unidade, um herói, uma construção, decorações, ou uma construção do reino.

| Propriedade | Tipo | Descrição   |
| ----------- | ---- | ----------  |
| x           | int  | Largura     |
| y           | int  | Comprimento |

## Exemplo de definição 

```json
{
    "width": 1,
    "height": 1
}
```

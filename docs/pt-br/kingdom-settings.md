# Configurações do Reino (Kingdom Settings)

Este arquivo define as propriedades globais do reino do jogador que não estão atreladas a uma entidade específica ou que servem de valor inicial (baseline).

| Propriedade              | Tipo | Descrição                                                                      | Exemplo de valor |
| ------------------------ | ---- | ------------------------------------------------------------------------------ | ---------------- |
| initial_occupation_limit | int  | O limite inicial de ocupação (habitação) que o reino suporta para as unidades. | 10               |

## Exemplo de definição

```json
{
    "initial_occupation_limit": 8
}
```

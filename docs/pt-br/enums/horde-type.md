# Tipo de horda

O tipo de horda determina o comportamento das unidades que surgem no campo de batalha em relação ao jogador e suas unidades.

| Nome     | Valor    | Descrição                                                                      |
| -------- | -------- | ------------------------------------------------------------------------------ |
| Pacífica | peaceful | Horda pacífica ou aliada, as unidades não atacam as unidades do jogador        |
| Hostil   | hostile  | Horda hostil, as unidades irão atacar as unidades do jogador e suas estruturas |

# Tipo de alvo

O tipo de alvo serve para definir se um feitiço, habilidade, unidade ou construção atinge um alvo único, uma área ou uma cadeia de alvos:

| Nome   | Valor  | Descrição                                                                                                             |
|------- | ------ | --------------------------------------------------------------------------------------------------------------------- |
| Único  | single | Atinge apenas um alvo, disponível para unidades, construções e feitiços                                               |
| Área   | area   | Atinge uma área, disponível para unidades, construções e feitiços; o valor de alcance deve ser ímpar para ter um meio |
| Cadeia | chain  | Atinge uma cadeia de alvos, disponível apenas para feitiços                                                           |

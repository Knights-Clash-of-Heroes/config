# Direção (Direction)

O enum de Direção define a orientação espacial dos objetos e o sentido de movimentação no mundo do jogo.

| Valor | Descrição | Correspondência Visual |
| ----- | --------- | ---------------------- |
| north | Norte     | Frente                 |
| south | Sul       | Atrás                  |
| east  | Leste     | Direita                |
| west  | Oeste     | Esquerda               |

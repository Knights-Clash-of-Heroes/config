# Tipo de custo

O tipo de custo serve para identificar o item requisitado:

| Nome    | Valor    | Descrição                                                               |
|-------- | -------- | ----------------------------------------------------------------------- |
| Recurso | resource | Recursos como ouro, cristais, madeira, pedra, ferro, energia e dragoons |
| Item    | item     | Itens como túnicas de couro, plantas de engenharia, alabardas, etc      |

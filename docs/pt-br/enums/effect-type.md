# Tipo de efeito

O tipo de efeito define como o impacto mecânico será aplicado.

| Nome       | Valor       | Descrição                                                                                             |
| ---------- | ----------- | ----------------------------------------------------------------------------------------------------- |
| Temporário | `temporary` | Dura um número determinado de turnos e afeta somente a unidade-alvo.                                  |
| Aura       | `aura`      | Ativa uma [Aura](../interfaces/aura.md) ao redor da unidade emissora durante um número de turnos. |

# Tipo de atributo

O tipo de atributo define o comportamento que o atributo exercerá sobre a unidade ou construção portadora.

| Nome        | Valor        | Descrição                                                                        |
| ----------- | ------------ | -------------------------------------------------------------------------------- |
| Mecânico    | `mechanical` | Aplica um valor numérico direto sobre uma stat da unidade                        |
| Aura        | `aura`       | Emite uma aura passiva e contínua ao redor da unidade enquanto ela estiver viva  |
| Descritivo  | `descriptive`| Sem impacto mecânico; serve apenas para exibição de informação (lore/visual)     |
